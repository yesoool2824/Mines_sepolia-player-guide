1. 준비물

① MetaMask 브라우저 확장 설치 & 지갑 생성
② Sepolia 네트워크 추가  →  MetaMask > 네트워크 드롭다운 > "Sepolia 테스트 네트워크" 선택
③ Sepolia ETH 받기  →  sepoliafaucet.com 또는 faucet.sepolia.dev 에서 무료 수령
④ 컨트랙트 주소  →  하우스(운영자)에게 받기

2. Remix에서 컨트랙트 연결

① remix.ethereum.org 접속
② 왼쪽 플러그인 탭 > Deploy & Run Transactions 클릭
③ Environment  →  Injected Provider - MetaMask 선택  (MetaMask 팝업에서 연결 승인)
④ Mines_2023100592.sol 파일을 Remix에 올리고 컴파일 (Ctrl+S)
⑤ Deploy 탭 하단 "At Address" 칸에 하우스에게 받은 주소 입력 → At Address 버튼 클릭
⚠️ "At Address" 버튼이 눌려야 기존 배포된 컨트랙트에 연결됨. 절대 Deploy 버튼 누르지 말 것.

3. Solo 모드 플레이

혼자서 하우스를 상대로 베팅하는 1인 게임.

① 게임 시작 & 베팅

write solo_requestGame 호출
  • _boardSize : 전체 칸 수 (하우스에게 확인)
  • _mineCount : 지뢰 수 (하우스에게 확인)
  • VALUE 칸 : 베팅할 ETH 금액 입력 후 호출
  → 반환된 게임 ID 메모 (이벤트 탭에서도 확인 가능)
⚠️ 하우스가 commitBoard를 완료해야 게임이 시작됨. 잠시 기다릴 것.

② 칸 열기

write solo_openTile(게임ID, 칸번호) 호출
  • 칸 번호는 0부터 시작 (0 ~ boardSize-1)
  → 하우스가 revealTile로 결과 공개 (이벤트 TileRevealed 확인)

③-A 지뢰 → 게임 오버

이벤트에 SoloBustedEvent 발생 → 베팅금 몰수. 게임 종료.

③-B 안전 → 계속 or 출금

선택방법결과
계속 플레이solo_openTile 재호출배당 배율 상승
중간 출금write solo_cashout(게임ID)현재 배율로 ETH 수령
현재 예상 배당 확인: read currentMultiplier(게임ID)
  → 반환값 ÷ 1,000,000 = 배율  (예: 1,100,000 → 1.1배)

4. Multi 모드 플레이

플레이어들이 ETH를 투표해 칸을 결정하는 다인 게임. 지뢰 칸에 투표한 사람만 돈을 잃음.

① 게임 참여

하우스에게 게임 ID 받기
write multi_joinGame(게임ID) 호출 (ETH 불필요)

② 투표로 칸 고르기

하우스가 투표 시작하면 (VotingStarted 이벤트):
write multi_voteOnTile(게임ID, 칸번호) + VALUE에 베팅 ETH 입력
  • 한 라운드에 하나의 칸에만 투표 가능
  • 투표 변경: multi_withdrawVote로 환불 후 재투표
  → 하우스가 multi_closeVoting → 가장 ETH 많이 몰린 칸 공개

③ 결과

결과내 투표 칸처리
안전 칸안전 칸에 투표safeBet으로 누적 (다음 라운드 계속)
안전 칸다른 칸에 투표ETH 즉시 환불
지뢰 칸지뢰 칸에 투표베팅금 몰수 (하우스 수입)
지뢰 칸다른 칸에 투표ETH 즉시 환불

④ 중간 출금 (선택)

write multi_cashout(게임ID)
  • 현재 누적 safeBet × 배율로 ETH 수령
  • 단, 보류 중인 투표 먼저 multi_withdrawVote로 회수할 것

5. 공통 — 타임아웃 보호

하우스가 30블록(약 6분) 이상 응답 없으면:
write claimTimeout(게임ID) → 베팅금 자동 환불
